# 📦 Mini Katalog Uygulaması

Flutter Günlük Eğitim Programı kapsamında geliştirilmiş, temel seviye bir **Mini Katalog Uygulamasıdır**. Harici paket kullanılmadan yalnızca `material.dart` altyapısıyla modern ve profesyonel bir UI tasarlanmıştır.

---

## 🚀 Özellikler & Öğrenim Çıktıları

- **Widget Ağacı & UI:** Temel widget'lar (`Container`, `Row`, `Column`, `Expanded`, `Padding` vb.) ile sıfırdan tasarım.
- **Navigasyon:** Sayfalar arası geçişlerde (`Navigator.push`, `Navigator.pop`) objeler üzerinden veri taşıma.
- **JSON & Data Modelleme:** `Product` model sınıfı ve `fromJson` metoduyla veri modelleme.
- **Listeleme:** `GridView` (`SliverGrid`) ve `ListView` yapılarıyla ürün sergileme.
- **Sepet Simülasyonu:** Local liste üzerinden Stateful widget yardımıyla sepete ürün ekleme ve sayaç güncelleme.

---

## 🛠️ Kullanılan Teknolojiler

| Alan | Teknoloji |
|------|-----------|
| Framework | Flutter 3.19.0+ |
| Dil | Dart 3.3.0+ |
| Mimari | Stateless / Stateful Widget |
| Paket | Yalnızca `material.dart` |

---

## ▶️ Çalıştırma Adımları

1. Projeyi sisteminize indirin.
2. Proje kök dizininde terminal veya komut satırı açın.
3. Paketleri yükleyin:
   ```bash
   flutter pub get
   ```
4. Bağlı bir telefon veya emülatörde başlatın:
   ```bash
   flutter run
   ```
